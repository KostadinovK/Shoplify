# Shoplify
A web platform for selling and buying all kind of stuff like electrical appliances, clothes, books and many more.

## Build with:
  * ASP.NET Core MVC
  * EF Core
  * Cloudinary
  * Stripe API for payments
  * SignalR
  * NUnit
  * ChartJS
  * Azure
